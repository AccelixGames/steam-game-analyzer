STEP 01

마케팅 포지셔닝 — 개발사의 의도

English Description

Fight, farm, build and work alongside mysterious creatures called "Pals" in this completely new multiplayer, open world survival and crafting game!

한국어 설명

드넓은 세계에 서식하는 신비한 생물 "팰"을 수집하여 전투, 건축, 농업에 투입하거나 공장에서 일을 시키는 등, 전에 없던 새로운 체험을 선사하는 멀티 지원 오픈월드 서바이벌 크래프트 게임입니다.

마케팅 키워드 분석

한 줄 피치: 영문 동사 4개(Fight, Farm, Build, Work) 병렬 나열 = "뭐든 할 수 있다". 한국어는 "수집→투입"이라는 지배-피지배 관계를 더 선명하게 드러내며, "공장에서 일을 시키는"이라는 노동착취 뉘앙스가 한국어에서 더 직접적.

전체 피치 구조: Q&A 형식, 12개 피처 개별 섹션. 소개 순서(=우선순위): ①생존 ②탑승/탐험 ③건축 ④생산 ⑤농업 ⑥공장/자동화 ⑦던전 ⑧브리딩 ⑨밀렵/범죄 ⑩멀티. "크리쳐 포획"은 독립 섹션 없음 — 모든 시스템의 전제 조건으로 암묵 처리.

Short vs Detailed 갭: 한 줄 피치에 없지만 전체 설명에서 강조: ①팰 도축/식용 ②밀렵/범죄 ③던전 ④브리딩/유전. "팰을 먹을 수 있다"는 상세 설명 도입부에 즉시 등장 — 다크 유머가 마케팅의 숨은 무기.

확정 보상

기지 자원 축적, 기술 해금, 팰 도감 진행, 기지 확장. 플레이할수록 가시적 성장.

가변 보상

브리딩 패시브 RNG, 레어 팰 출현, 던전 드롭. "골든 패시브 4개" 조합은 수십 시간 배합의 가챠적 쾌감.

팰의 이중 역할(전투 동료+기지 노동력)이 두 루프를 연결하는 핵심 축. 포획 하나로 전투력과 생산력이 동시 증가하는 이중 보상 구조. 자동화 진행 시 역할이 "생존자"→"경영자/수집가"로 전환되며, "주객전도" 밈의 원천.
